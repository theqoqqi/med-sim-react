
КАК ФИКСИТЬ БАГ С ПРОКРУТКОЙ И ФЛЕКСОМ:
На самом верхнем диве ОБЯЗАТЕЛЬНО должно быть указано display: flex и высота НЕ в процентах (vh, px)
На каждом промежуточном диве ОБЯЗАТЕЛЬНО должно быть указано display: flex и height или min-height: 0 (или больше)

Подробнее в папке P:\OVERFLOW + FLEX FIX



Добавить списки имен и выбор случайного имени.
Добавить пациентам заметки, чтобы можно было к каждому пациенту приписать любой текст с заметками для себя.
Добавить кнопку "прийти через N дней" (как вариант, просто выбор от 0 до N рядом с кнопкой "Отправить домой").
Добавить возможность фильтровать список препаратов.



Возможно, стоит добавить симптомы. Как вариант, можно сделать их с помощью числовых параметров. Или все же отдельно.

Можно добавить хронические болезни, которые нельзя вылечить.
Можно добавить болезням постоянный эффект после болезни (или, вернее, чтобы они могли становиться хроническими).
Возможно это можно сделать, просто добавив некоторым болезням свойство типа "хроническийМножитель".
Таким образом, закроются сразу оба пункта, и можно будет настраивать силу хронического варианта.
Возможно, нужно будет добавить еще шанс перехода в хроническое заболевание.

Можно добавить болезням побочные эффекты, проявляющиеся с определенным шансом.
